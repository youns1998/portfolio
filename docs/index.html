Project: 여행 경로 플래너

지도 기반 여행 계획 서비스

🔗 구현 보기

여행 경로 플래너는 사용자가 쉽게 여행 일정을 계획하고 관리할 수 있는 웹 서비스입니다. 카카오 지도 API를 활용하여 여행지를 검색하고 경로를 시각화하여 효율적인 여행 계획을 수립할 수 있습니다.

직관적인 드래그 앤 드롭 인터페이스로 일정을 관리하고, 관광지 정보, 이동 시간, 거리 계산 등 실질적인 여행 계획에 필요한 정보를 제공합니다. 한국관광공사 오픈 API를 연동하여 신뢰할 수 있는 관광지 정보를 제공합니다.

기술 스택
HTML/CSS JavaScript jQuery 카카오 지도 API Bootstrap 5 한국관광공사 API Axios
주요 기능
  • 지도 기반 여행지 검색 및 일정 설계
  • 드래그 앤 드롭으로 일정 순서 변경
  • 이동 거리 및 소요 시간 자동 계산
  • 관광지/맛집/숙소 카테고리별 정보 제공
  • 일정 저장 및 PDF 출력 기능
  • 모바일 반응형 디자인
개발 포인트
  • 카카오 지도 API를 활용한 지도 렌더링 및 마커 처리
  • 외부 관광 데이터 연동 및 효율적인 데이터 캐싱
  • 사용자 위치 기반 주변 정보 검색 기능 구현
  • LocalStorage를 활용한 사용자 데이터 관리
  • 크로스 브라우저 호환성 및 모바일 최적화

Project: 회원 관리 시스템

Spring Boot 기반 회원 관리

🔗 구현 보기

회원 관리 시스템은 Spring Boot 프레임워크를 활용한 안정적인 회원 정보 관리 웹 애플리케이션입니다. 회원 가입, 로그인, 프로필 관리, 권한 설정 등 기본적인 회원 관리 기능을 제공합니다.

이 프로젝트는 보안이 중요한 회원 정보를 안전하게 관리하는 것에 중점을 두었습니다. Spring Security를 활용하여 JWT 기반 인증 시스템을 구현했으며, JPA를 이용해 데이터를 효율적으로 관리하고 RESTful API 설계 원칙을 준수하였습니다.

기술 스택
Java 11 Spring Boot 2.6 Spring Security MySQL 8.0 JPA/Hibernate JWT Maven Thymeleaf
주요 기능
  • 이메일 인증 회원가입 및 소셜 로그인
  • JWT 기반 인증 및 권한 관리
  • 회원 정보 CRUD 기능
  • 관리자 대시보드
  • 사용자 활동 로그 기록
  • 비밀번호 암호화 및 보안 강화
개발 포인트
  • 계층형 아키텍처(Controller-Service-Repository) 설계
  • Spring Security와 JWT를 활용한 안전한 인증 시스템 구현
  • JPA를 활용한 효율적인 데이터베이스 연동
  • 단위 테스트 및 통합 테스트 작성
  • RESTful API 문서화 (Swagger)

Project: 실시간 화상 회의

WebRTC 기반 화상 회의

🔗 구현 보기

실시간 화상 회의는 WebRTC 기술을 활용한 브라우저 기반 화상 회의 서비스입니다. 별도의 플러그인 설치 없이 웹 브라우저에서 바로 화상 통화와 화면 공유가 가능합니다.

이 프로젝트에서는 Node.js와 Socket.IO를 활용하여 시그널링 서버를 구축하고, WebRTC의 P2P 연결을 통해 저지연 고품질 화상 통화를 구현했습니다. 또한 채팅, 화면 공유, 회의실 생성 등의 기능을 제공하여 실용적인 화상 회의 솔루션을 개발했습니다.

기술 스택
HTML/CSS JavaScript WebRTC Node.js Socket.IO Express.js AWS EC2
주요 기능
  • 실시간 화상 통화 (최대 4인)
  • 화면 공유 및 녹화 기능
  • 실시간 채팅
  • 회의실 ID 생성 및 참여
  • 마이크/카메라 설정
  • 모바일 웹 지원
개발 포인트
  • WebRTC를 활용한 P2P 연결 구현
  • Socket.IO를 이용한 실시간 시그널링 서버 개발
  • 미디어 스트림 처리 및 최적화
  • 네트워크 상태에 따른 적응형 화질 조정
  • AWS EC2 인스턴스에 서비스 배포
